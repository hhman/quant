# T2: 去极值方法选择（MAD vs 3σ）- 需求文档

## 📋 需求概述

### 需求背景
当前框架使用3σ方法进行因子去极值处理，而华泰多因子系列2标准使用MAD（中位数绝对偏差）方法。虽然`core/factor_analysis.py`中已实现`ext_out_mad`函数，但Step1实际调用的是`ext_out_3std`。

### 需求目标
1. ✅ 将默认去极值方法切换为MAD（对齐华泰标准）
2. ✅ 通过CLI参数`--outlier-method`支持方法切换（mad | 3sigma）

### 设计原则
- **默认对齐原则**：默认方法为MAD
- **向后兼容原则**：保留3σ方法作为可选项

#### 修改点2：去极值处理逻辑改造

**位置**：去极值处理部分（约Line 94-98）

**原代码**：
```python
print("  执行去极值处理...")
factor_df = factor_df.groupby(level="datetime").apply(
    lambda x: ext_out_3std(x, valid_factor_cols)
)
```

**修改为**：
```python
print(f"  执行去极值处理 (method={outlier_method})...")

if outlier_method == 'mad':
    factor_df = factor_df.groupby(level="datetime").apply(
        lambda x: ext_out_mad(x, valid_factor_cols)
    )
    print(f"  ✓ 使用MAD方法（中位数绝对偏差）")
elif outlier_method == '3sigma':
    factor_df = factor_df.groupby(level="datetime").apply(
        lambda x: ext_out_3std(x, valid_factor_cols)
    )
    print(f"  ✓ 使用3σ方法（3倍标准差）")
else:
    raise ValueError(f"未知的去极值方法: {outlier_method}")
```

## 📊 方法对比

| 维度 | MAD方法 | 3σ方法 |
|-----|---------|--------|
| **统计量** | 中位数 + 中位数绝对偏差 | 均值 + 标准差 |
| **稳健性** | ✅ 对偏态分布稳健 | ❌ 假设正态分布 |
| **阈值计算** | `median ± 3 × MAD` | `mean ± 3 × std` |
| **论文标准** | ✅ 华泰多因子系列2 | ❌ 非华泰标准 |
