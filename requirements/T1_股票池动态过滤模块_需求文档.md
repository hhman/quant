# T1: 股票池数据基础建设 - 需求文档

## 📋 需求概述

### 需求背景
当前框架在数据预处理阶段缺少股票可交易性相关的标记数据，导致策略构建阶段无法灵活应用股票池过滤。根据华泰多因子系列2标准，策略层需要过滤以下股票：
- ST股票（包括*ST）→ 风险溢价污染
- 新股（上市不足60天）→ 流动性偏差
- 停牌股票 → 无法实际交易

### 需求目标
在Step0数据预处理阶段，计算并导出股票可交易性相关标记，为策略层提供数据基础：
1. ✅ 识别ST股票区间 → 生成 `instruments/st.txt`
2. ✅ 计算上市天数 → 添加到CSV `listing_days` 列
3. ✅ 保留成交量数据 → 已有 `volume` 列

### 设计原则
- **最小侵入原则**：修改仅涉及Step0，Step1-4无需改动
- **数据基础原则**：Step0提供标记数据，策略层灵活使用
- **因子研究纯粹性**：因子计算使用市场股票池，不过滤
- **避免幸存者偏差**：收益率数据使用全市场（已完成）

```
└─────────────────────────────────────────────────────────┘
                          ↓

│ ────────────────────────────────────────────────────     │

│ • 因子提取 → 中性化 → 收益计算 → 绩效评估                │
│ • 不应用股票池过滤（保持统计纯粹性）                      │
```

### 设计决策

#### 决策1：因子研究 ≠ 策略构建
**规则**：因子研究阶段不应用股票池过滤
- **因子研究目的**：评估因子的统计显著性和预测能力
  - IC、Rank IC 需要包含全市场股票（包括ST、新股）
  - 因子收益回归需要客观评估因子值与收益的关系
  - 人为过滤会引入偏差，扭曲因子的真实表现

- **策略构建目的**：构建可执行的投资组合
  - ST股票风险溢价，不应纳入策略股票池
  - 新股流动性不足，需要过滤
  - 停牌股票无法交易，必须剔除

**结论**：
- ✅ Step0-4（因子研究层）：不应用过滤，使用市场股票池
- ✅ 策略层：根据实际交易约束应用过滤

#### 决策2：ST股票的识别时机 ✅ 已明确
**规则**：ST股票在T+1日识别
- T日：交易所公布ST名单，股票名称不变
- T+1日：股票名称变更为"ST"，Step0检测到并记录
- 结果：T日的标记为正常，T+1日及以后标记为ST
- 符合实盘逻辑

#### 决策3：新股上市日的计数 ✅ 已明确
**规则**：上市当天算第0天
```python
listing_days = (交易日期 - 上市日).days
# 上市日: listing_days = 0
# 次日:   listing_days = 1
...
# 策略层过滤条件: listing_days < 60（即前60天都剔除）
```

#### 决策4：停牌股票的判断 ✅ 已明确
**规则**：`volume = 0` 即为停牌
- 简单直接，无需额外校验
- 策略层可使用此标记过滤

**文件格式**：

**说明**：

## 🚨 注意事项

### 数据可用性
**注意**：Step0生成的标记数据通过Qlib导出后，策略层可通过以下方式访问：
```python
# 读取 ST 标记
D.features(instruments, fields=["$ST股票"], ...)

# 读取上市天数
D.features(instruments, fields=["$listing_days"], ...)

# 读取成交量
D.features(instruments, fields=["$volume"], ...)
```

### 因子研究 vs 策略构建
**重要区分**：
- **因子研究（Step1-4）**：
  - 目的：评估因子的统计显著性和预测能力
  - 股票池：使用市场定义的股票池（如 csi300、all）
  - 过滤：不应用任何过滤

- **策略构建（未来实现）**：
  - 目的：构建可执行的投资组合
  - 股票池：基于市场股票池，应用可交易性过滤
  - 过滤：ST、新股（<60天）、停牌

### 幸存者偏差处理
**已解决**：收益率数据使用全市场（`market="all"`）
- 已通过跨市场复用优化完成
- 退市股票的收益率包含在全市场数据中
- Step3通过 `join` 自动对齐因子和收益率数据
